# # ProfMatch - Интерактивный прототип

## Описание проекта
Прототип информационной системы для рекрутингового агентства «ProfMatch». 
Система предназначена для автоматизации процессов подбора персонала: 
управление вакансиями, ведение базы кандидатов, поиск и привязка кандидатов, 
планирование собеседований.

## Структура репозитория

| Папка | Описание |
|:---|:---|
| `/wireframes` | Наброски wireframe для всех экранов |
| `/user-flow` | Диаграммы пользовательских потоков |
| `/task-flow` | Диаграммы последовательности и активности |
| `/interactive-prototype` | Ссылки на интерактивный прототип (Figma) |
| `/docs` | Документация (screenflow, требования) |

## Ветки
- `main` - основная ветка (стабильная версия)
- `feature/user-flow-prototype` - ветка для разработки прототипа
